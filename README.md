# S1UI-mod-kr
TERA UI mod for KR server

## Installation
- Make directory at TERA/S1Game/CookedPC called `_mod`
- Unzip wanted client mods into `_mod` folder
- [Preview](https://seraphinush-gaming.tumblr.com/teramod)
