# client-mod
TERA S1UI client mod

## Installation
- Make directory at TERA/S1Game/CookedPC called `_mod`
- Unzip wanted client mods into `_mod` folder
- [Preview](https://github.com/seraphinush-gaming/client-mod/wiki/TERA-client-mod)
